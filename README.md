# Babel Webpack Starter

MicroPosts : Formation https://www.udemy.com/course/modern-javascript-from-the-beginning

### Version
1.1.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

```sh
$ npm run json:server
```

### Build
Compile and build

```sh
$ npm run build
```
